## Objetivo
Implementar un editor de texto simple en consola que permita realizar operaciones de escritura, deshacer (Undo) y rehacer (Redo) utilizando la estructura de datos pila.

## Concepto de Pila
Una pila es una estructura de datos que sigue el principio LIFO (Last In, First Out). Las operaciones principales son push, pop, peek e isEmpty.

## Funcionamiento
El programa utiliza dos pilas:

1. Pila de acciones → guarda las acciones realizadas
2. Pila de redo → guarda las acciones deshechas

Esto permite implementar las funciones de deshacer y rehacer.
